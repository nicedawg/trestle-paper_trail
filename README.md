# Trestle::PaperTrail

Easily display versioning info from PaperTrail within your admin resources.

## Installation
Add this line to your application's Gemfile:

```ruby
gem "trestle-paper_trail"
```

And then execute:
```bash
$ bundle
```

Or install it yourself as:
```bash
$ gem install trestle-paper_trail
```
